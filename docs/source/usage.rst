Usage
=====

Démarrage de l'application
--------------------------

Pour démarrer, il suffit de se rendre dans le répertoire 'planningpoker' et exécuter la commande suivante :

.. code-block:: console

   (.venv) $ npm run dev

Ensuite, on se rend dans le répertoire 'back' et on exécute la commande suivante :

.. code-block:: console

   (.venv) $ uvicorn main:app --host 127.0.0.1 --port 8000
